# Caixa de Entrada

Esta pasta contém ideias recém-capturadas com estado **Capturada**.

A captura deve ser rápida. Não é necessário preencher toda a análise antes de salvar a ideia.

Quando a ideia passar para **Em análise**, **Incubada** ou **Pronta para testar**, mova o arquivo para `ideas/` e atualize o `INDEX.md`.
